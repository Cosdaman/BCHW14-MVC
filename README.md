# BCHW14-MVC  

## Description  

This is a tech-blog that features user authentication, password hashing, and database usage. The website is deployed on Heroku, and uses JawsDB MySQL2 as its database.

## Technologies Used  
- NodeJS
- bcrypt
- connect-session-sequelize
- dotenv
- express
- handlebars
- mysql2
- sequelize
- HTML
- CSS
- Javascript
